# E6Data Design System

Welcome to the E6Data Design System documentation.

## Overview

This design system provides a consistent set of React components and design tokens
used across E6Data applications (Console and Product).

## Key Features

- **Consistent Design Tokens**: Shared color palette, typography, and spacing
- **Radix UI Foundation**: Accessible, unstyled primitives
- **Tailwind CSS**: Utility-first styling approach
- **Dark Mode Support**: Full dark mode via CSS custom properties
- **Sharp Aesthetic**: Zero border radius for a modern, distinctive look

## Getting Started

```tsx
import { Button, Card } from "e6ds";
import "e6ds/styles";

function App() {
  return (
    <Card>
      <Button>Get Started</Button>
    </Card>
  );
}
```

## Component Categories

- **Primitives**: Button, Card, Input, Select, Dialog, etc.
- **Forms**: FormField, Checkbox, RadioGroup, Switch, etc.
- **Layout**: Sidebar, TopBar, MainContent
- **Feedback**: Toast, Alert, Progress, ConfirmDialog
